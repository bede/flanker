# Installation

## From GitHub

Ensure you have all dependencies installed:

**Python dependencies:**

* multiprocessing
* sys
* argparse
* pandas
* numpy
* subprocess
* biopython
* pathlib
* time
* logging
* tempfile
* os
* collections
* glob
* networkx

**External software:**

* abricate
* mash

Then simply clone the repository:

```
  git clone https://github.com/wtmatlock/flanker
```

and check everything is working:

```
  python flanker.py --help
```
